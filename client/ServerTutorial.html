Drawingly Server Tutorial

The two main drives of Drawingly project are making a "portable" and "customizable" game. This tutorial will show how to start your own Drawingly game server and how to customize it as you want.


Starting the Server

Starting your own game server is simple in Drawingly.
  1. Check if Java is installed on your system. If not, download and install from its official page.
  2. Download this zip file that contains the server folder.
  3. Extract the contents of the zip file to a folder: "DrawinglyServer.jar, StartServer.bat, StartServer.sh, config.json, words.txt"
  4. Make sure that the PORT number that is writing in "config.json" file (on the first line) is open for your router. (See these pages for more info.)
  5. For Windows, run "StartServer.bat" file, for Linux/Mac run "StartServer.sh" file.
  6. Share the IP address of your computer and the PORT number that you have set in your "config.json" file to the players that will connect to your server.

Customizing the Server

Before starting the server, you can configure the game that you want to run on the server.

  • By editing "config.json" file that is located in the server folder, you can modify some of the parameters about the game. (turn timer, guessing/drawing scores, maximum number of players, etc.)
  • You can specify the words that will come up in the game, by listing them in the file "words.txt".
